7-Joint Optimized
Acrylic Truss

MATLAB Structural Analysis Buckling Analysis Iterative Optimization Material Constraints

Overview

Maximum load capacity from minimum material

Given a fixed budget of acrylic strip material, a set of permitted joint configurations, and a load application point, the challenge was to design a 7-joint truss that maximizes the load it can hold before catastrophic failure — then build it and see if the model was right.

This project demanded close integration between computational modeling and physical intuition. The final structure supported 55.38 oz, exceeding the optimized prediction of 53.07 oz, and the MATLAB model correctly identified which member would buckle first under increasing load.

Methodology

Structural modeling & iterative geometry refinement

I implemented a full 2D truss solver in MATLAB, assembling the global stiffness matrix via the direct stiffness method. Member forces were extracted under a unit applied load, then scaled to predict failure loads using Euler buckling theory for the compression members and tensile yield stress for members in tension.

The initial design predicted a capacity of 38 oz. Through geometric iteration — adjusting joint positions while respecting the 7-joint constraint — I identified that shallow-angle compression diagonals were the bottleneck. Steepening those members reduced effective slenderness ratio and raised the predicted capacity to 53.07 oz, a 40% improvement over the baseline.

The physical build used laser-cut acrylic strips joined with precision-drilled pin connections. Alignment fixtures were fabricated to ensure joint geometry matched the model within 0.5 mm.

Results

Model validated by physical test

  • Initial design predicted 38 oz capacity; optimized design predicted 53.07 oz
  • Final physical structure held 55.38 oz — 4.4% above prediction
  • Critical buckling member correctly identified by MATLAB model prior to testing
  • 40% load capacity improvement achieved through geometric optimization alone
  • Model accuracy sufficient for engineering design confidence
